## Тестовое задание: реализация backend-приложения на Django с интеграцией Stripe для оплаты товаров

**Demo:** https://stripe-store-app.onrender.com (открывается с задержкой, потому что тариф хостинга бесплатный)


### Реализованные фичи
- Добавление товаров и просмотр заказов через админ-панель Django
- Добавление товаров в корзину (Django sessions) с последующей оплатой одной транзакцией через Stripe Checkout или оплата только одного товара
- Поддержка нескольких валют (на данный момент Евро и Доллары)
- Добавлены модели Discount и Tax, настраиваются в админ-панели
- Реализован деплой через Docker и docker-compose
- Автоматическое применение миграций и создание админ-пользователя при деплое

### Список эндпоинтов
Пользовательские:
- `GET /item/<id>/` - страница товара
- `GET /cart/` - cтраница корзины
- `GET /admin/` - админ-панель

Служебные:
- `GET /cart/add/<item_id>/` - добавление товара в корзину
- `GET /checkout/` - оплата корзины через Stripe Checkout
- `GET /buy/<id>` - опалата конкретного товара через Stripe Checkout


### Локальный запуск
1. Создать файл `.env` по примеру `.env.example`. Stripe ключи необходимо взять в [Stripe Dashboard](https://dashboard.stripe.com/)
2. Запустить приложение через Docker:
```shell
docker compose up --build -d
```
3. Приложение будет доступно по адресу http://localhost:8000

Для успешной тестовой оплаты использовать следующие данные карты:
```shell
4242 4242 4242 4242
Любая будущая дата
Любой CVC
```
